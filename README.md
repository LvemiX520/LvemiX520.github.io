# LvemiX520.github.io

个人博客网站项目，基于GitHub Pages部署。

## 主要功能

- 个人博客文章展示
- 相册展示
- 音乐播放
- 留言板功能
- 文章分类和标签
- 关于页面
- 隐私政策页面

## 目录结构

```
├── 404.html                # 404页面
├── CNAME                   # 自定义域名配置
├── Cookies/                # Cookies相关页面
├── LX.ico                  # 网站图标
├── MessageBoard/           # 留言板
├── about/                  # 关于页面
├── air-conditioner/        # 空调相关页面
├── album/                  # 相册
├── anzhiyu/                # Anzhiyu主题相关文件
├── archives/               # 文章归档
├── assets/                 # 静态资源
├── bangumis/               # 番剧页面
├── categories/             # 文章分类
├── copyright/              # 版权声明
├── css/                    # CSS样式文件
├── essay/                  # 随笔页面
├── gallery/                # 画廊
├── img/                    # 图片资源
├── index.html              # 首页
├── js/                     # JavaScript文件
├── link/                   # 友情链接
├── music/                  # 音乐页面
├── posts/                  # 文章内容
├── privacy/                # 隐私政策
├── search.xml              # 搜索索引
├── tags/                   # 文章标签
└── updateLog/              # 更新日志
```

## 使用说明

1. 克隆本仓库
2. 修改配置文件
3. 添加文章到`posts`目录
4. 提交更改并推送到GitHub

## 部署

本项目通过GitHub Pages自动部署，访问地址：<https://lvemix520.github.io>
